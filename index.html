My Projects

East Java Provincial Health Index Model Using Ridge Regression

This project aims to overcome the problem of multicollinearity that occurs in the regression model using the Ridge Regression method with the East Java Provincial Health Index Model.

Comparison of Partial Least Square Regression and Principal Component Regression methods in overcoming multicollinearity problems

This project aims to compare the Partial Least Square Regression and Principal Component Regression methods in overcoming multicollinearity problems. The case study used is the influence of education and the field of inequality and poverty on the East Java Province Education Index

Comparison of Arima and Double Exponential Smoothing Methods in Forecasting the Price of Cayenne Chili in the Wonokromo Market in Surabaya City

This project aims to compare the Arima and Double Exponential Smoothing methods in predicting the price of cayenne pepper in the Wonokromo Market in Surabaya for the next 5 months and comparing the effectiveness of forecasting the price of cayenne pepper

Food price forecasting in East Java Province by using the Arima method

This project aims to predict food prices in East Java Province using the Arima method. The data used in this study are time series data from 2018 to 2022 obtained from the website of the National Strategic Food Price Information Center (PIHPS) website
